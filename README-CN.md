# Nano Fast Api Framework

Nano 是一款零配置、无骨架、极小化的 Hyperf 发行版，本项目基于 Nano 进行二次封装， 可以快速开发高并发、快响应的api应用。   

## 功能

* 路由
* 数据模型
* 单入口
* 中间件
* 国际化
* 日志
* ...

## 启动

```bash
php bin/nano start
```
